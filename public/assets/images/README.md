- Optimize for web (use tools like ImageOptim or sharp). Prefer 2x for retina.
- Keep file sizes reasonable for fast load (<300KB per hero image).

Replace references in components/sections with your actual filenames.
If a file is missing, the site gracefully falls back to 3D or gradients.
